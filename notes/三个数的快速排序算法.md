# 三个数的快速排序算法

这个方法也是冒泡算法的思想。**排序至从小到大**

也就是运用三个连续的`if` 来实现：

*若有 a, b, c 三个数，swap 作为暂存*

```c
if (a > b) {
    sawp = a;
    a = b;
    b = swap;
}
if (b > c) {
    swap = b;
    b = c;
    c = swap;
}
if (a > c) {
    swap = a;
    a = c;
    c = swap;
}
```

大致思想就是从左到右依次两个两个比较，然后再回来比较第一项和第三项。
